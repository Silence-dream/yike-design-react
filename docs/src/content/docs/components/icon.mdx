# icon 图标

在这汇总Yike Design 内置的图标，将不定期更新。

## 使用方法

使用图标组件，你需要安装 `@yike-design/react-icons` 图标组件包：

```shell pure
pnpm add @yike-design/react-icons
```

## 图标列表

<IconList />

## 基本使用

从 `@yike-design/react-icons` 中引入需要的图标，然后直接使用即可

```tsx
import { Space } from '@yike-design/react';
import { AvatarOutlined, TickCircleOutlined } from '@yike-design/react-icons';

export default () => {
  return (
    <Space>
      <AvatarOutlined />
      <TickCircleOutlined />
    </Space>
  );
};
```

## 旋转角度

通过 `rotate` 属性设置图标旋转角度，单位为度数。

```tsx
import { Space } from '@yike-design/react';
import { WordOutlined } from '@yike-design/react-icons';

export default () => {
  return (
    <Space>
      <WordOutlined />
      <WordOutlined rotate={45} />
      <WordOutlined rotate={90} />
      <WordOutlined rotate={180} />
    </Space>
  );
};
```

## API

除了一下几个属性，其他可以设置给 span 元素的属性都可以设置

| 参数      | 描述             | 类型                       | 默认值 |
| --------- | ---------------- | -------------------------- | ------ |
| className | 设置图标的样式名 | string                     | -      |
| rotate    | 图标旋转角度     | number                     | 0      |
| svgRef    | 获取 svg 元素    | `React.Ref<SVGSVGElement>` | -      |
